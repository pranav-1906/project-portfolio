# ⚡ Project Portfolio (v2.0)

> "Half designer, half developer, fully caffeinated." ☕

Welcome to the repository for my newly redesigned personal portfolio. As a UI/UX Designer and Full-Stack Developer, I wanted a digital space that reflects both my creative eye and my engineering logic. 

This v2.0 upgrade strips away the bulky custom CSS of my previous site in favor of a sleek, utility-first **Tailwind CSS** architecture, wrapped in a modern "Tech Brutalism meets Glassmorphism" aesthetic.

🔗 **[View Live Site Here](https://pranav19.netlify.app/)**

---

## 🚀 What's New in v2.0?

* **Tailwind CSS Overhaul:** Transitioned from 600+ lines of vanilla CSS to a highly maintainable, utility-first approach using the Tailwind CDN.
* **Interactive Glassmorphism:** Implemented frosted glass panels (`backdrop-blur`) that dynamically slide over animated, fixed ambient background orbs for true depth.
* **The Bento Box Arsenal:** Redesigned the skills section into a responsive, highly visual Bento Grid separating Web Dev, Design, AI & Automation, and Databases.
* **Mac OS Terminal Contact:** Bypassed standard boring forms with a functional, terminal-themed interface that compiles user input and launches their native email client via vanilla JavaScript.
* **Smart Floating Navigation:** Engineered a mobile-first floating dock that hides on scroll-down to maximize screen real estate and smoothly slides back up on scroll-up.
* **Tactile Dark Mode:** Built a custom "pull-cord" theme toggle using Tailwind's `group` utility classes and `localStorage` for persistent preference saving.

## 🛠️ Tech Stack

* **Markup:** HTML5
* **Styling:** Tailwind CSS 
* **Scripting:** Vanilla JavaScript (ES6+)
* **Typography:** [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk)
* **Deployment:** Netlify

## 💻 Local Development

No complex build steps required. To run this locally:

1. Clone the repository:
   ```bash
   git clone [https://github.com/pranav-1906/project-portfolio.git](https://github.com/pranav-1906/project-portfolio.git)
   ```
2. Open the directory.

3. Launch index.html in your browser or spin it up using VS Code Live Server.
